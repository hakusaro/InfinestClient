Keys:
T acts like a hammer, to make slopes faster
F ports you to the cursor
Alt destroy tiles or walls at cursor position (you can select which you want to destroy in the last page of the ui)
Ctrl place currently selected block at cursor position (use an empty bucket to remove any liquid, water buckets to place water, lava buckets to place lava, honey buckets to place honey)
Tab shows FPS and playerlist
R converts ground to the biome set with .con (also works in multiplayer, if not protected)
Middle Mousebutton (hold) Walk trough walls
